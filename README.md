# 🏠 おうち倉庫

シンプルで使いやすい日用品の在庫管理PWA（Progressive Web App）です。

## 特徴

- ✨ シンプルで直感的なUI
- 📱 PWA対応 - ホーム画面に追加してアプリのように使用可能
- 🔌 オフライン対応 - インターネット接続なしでも動作
- 💾 ブラウザのLocalStorageを使用したデータ永続化
- 📊 在庫状態の自動判定（在庫あり/在庫不足/在庫切れ）
- 🔮 消費予測機能 - 過去の使用履歴から在庫切れ日を予測
- 💰 価格追跡 - 商品価格の変動を記録・表示
- 🛒 ショッピングリンク - Amazon・楽天への直接リンク
- 🔍 カテゴリ別フィルタリング
- ⚠️ 在庫不足アラート表示
- 📱 レスポンシブデザイン対応（モバイル最適化）

## 使い方

### 初回セットアップ

1. ブラウザで `index.html` を開きます（HTTPSサーバーでホストすることを推奨）
2. PWAとしてインストール（任意）:
   - **Chrome/Edge (PC)**: アドレスバー右側の「インストール」ボタンをクリック
   - **Safari (iOS)**: 共有ボタン → 「ホーム画面に追加」
   - **Chrome (Android)**: メニュー → 「ホーム画面に追加」

### 基本操作

1. 「新しい日用品を追加」フォームから商品を登録します
   - **商品名**: 管理したい日用品の名前（例：トイレットペーパー）
   - **在庫数**: 現在の在庫数
   - **最低在庫数**: この数以下になると「在庫不足」として警告されます
   - **参考価格**: 商品の価格（任意・価格変動追跡に使用）
   - **カテゴリ**: 商品のカテゴリを選択
   - **購入リンク**: Amazon/楽天の商品URLを登録（任意）
2. 登録した商品は下部の一覧に表示されます
3. 「＋」「−」ボタンで在庫数を増減できます
4. 「削除」ボタンで商品を削除できます
5. 価格の✏️ボタンで価格を更新できます

## 機能

### 在庫管理
- 商品の追加・削除
- 在庫数の増減（＋/−ボタン）
- 在庫状態の自動判定
  - **在庫あり**: 在庫数が最低在庫数を上回っている
  - **在庫不足**: 在庫数が最低在庫数以下
  - **在庫切れ**: 在庫数が0

### 消費予測機能 🔮
- 過去30日間の使用履歴から消費ペースを自動計算
- 在庫切れまでの日数を予測表示
- 3日以内に在庫切れが予測される商品を警告バナーで通知
- 最低2回以上の減少記録が必要

### 価格追跡機能 💰
- 商品登録時の価格を記録
- 価格変更時に初回価格との差額・変動率を表示
- 値上がり/値下がりを色分けして視覚化
  - 🔴 赤: 値上がり
  - 🟢 緑: 値下がり

### ショッピングリンク 🛒
- 商品名でAmazon/楽天を検索
- 購入リンクを登録して商品カードから直接アクセス
- ワンクリックで商品購入ページへ遷移

### フィルタリング
- カテゴリ別表示（日用品、食品、洗剤、衛生用品、その他）
- 在庫不足のみ表示

### データ保存
- ブラウザのLocalStorageを使用
- ページを閉じてもデータは保持されます
- ブラウザのキャッシュをクリアすると削除されます
- 約500商品以上の登録が可能（5-10MB容量）

### PWA機能
- **オフライン対応**: インターネット接続なしでも利用可能
- **ホーム画面追加**: スマートフォンのホーム画面にアプリとして追加
- **高速起動**: Service Workerによるキャッシュで即座に起動
- **自動更新**: 新しいバージョンが利用可能になると自動通知

## PWAアイコンの準備

PWAとして完全に動作させるには、アイコン画像が必要です。

### 方法1: オンラインツールを使用（推奨）

1. [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) にアクセス
2. 512x512px以上のアイコン画像をアップロード
3. 生成されたアイコンをダウンロードして `icons/` ディレクトリに配置

### 方法2: 自動生成スクリプトを使用

```bash
cd icons
./generate-icons.sh your-icon-image.png
```

必要なアイコンサイズ：
- 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512

詳細は `icons/README.md` を参照してください。
